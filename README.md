# Expense Tracker (CLI)
A command-line based expense tracker built using Python.

## Features
- Add expenses with category, amount, and date
- View all expenses
- Filter expenses by category
- Show total spending
- Delete expenses
- Persistent storage using file handling

## Tech Used
- Python
- File Handling

## How to Run
1. Clone the repository
2. Run main.py
